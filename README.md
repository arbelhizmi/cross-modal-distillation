# cross-modal-distillation

[Project Page](https://cross-modal-distillation.github.io/) | [arXiv](https://arxiv.org/abs/2604.09076)

Official implementation of the paper "Cross-Modal Knowledge Distillation from Spatial Transcriptomics to Histology".

### Setup
Our code was developed using Python 3.10. We recommend using a virtual environment and installing the packages listed in `environment.yaml`
